# docs

项目文档与团队规范统一放在这里，作为前后端协作和 AI 辅助开发的共同依据。

## 已归档规范

- `GIT_GUIDELINES.md`: Git 使用规范，覆盖分支、提交、同步、PR、Review、发布、Hotfix 与不入库清单。
- `API_GUIDELINES.md`: REST、WebSocket、数据结构与接口变更规则。
- `../rules.md`: VibeCoding 使用规则，约束 AI 辅助开发的工作流、安全红线和输出格式。

## 本仓补充文档

- `PROJECT_STRUCTURE.md`: 当前项目目录架构与模块职责。
- `PROMPTS.md`: 团队可复用 Prompt 模板。
