# 调度控制台使用说明

调度控制台位于路网大屏 `/`，用于查看展示数据和仿真状态，并操作前端演示功能。

## 使用顺序

1. 查看右上角数据来源。“数据库数据”表示 dashboard bootstrap 成功；“本地演示”表示使用 mock。该标识不等同于实时仿真连接状态。
2. 查看交通统计、中央路网和告警。
3. 选择路口，在“AI 信号控制”查看当前前端状态和建议。
4. 需要实时仿真时创建并启动仿真，由 WebSocket 帧更新车辆、道路、路口和信号。
5. 使用应急调度表单选择起终点、车辆类型和优先级。
6. 使用 Agent 查询状态或获取解释；没有工具证据时不能把回答当作实时事实。

## 控制边界

- “启动 AI 自适应控制”“暂停 AI 控制”和“手动切换相位”只修改前端 store。
- 应急调度存在后端 `sid` 时优先调用后端，失败时回落本地模拟。
- 告警确认只修改前端确认状态。
- 策略对比指标可能来自 bootstrap 或 mock，不是自动生成的实验结论。
- 当前没有角色授权、审批或真实信号机控制闭环。

因此，控制台可以用于演示、联调和仿真观察，不能据其页面按钮状态声称已控制真实路口。
