# 调度控制台使用说明

## 文档信息

- 知识库分类：系统操作手册知识库
- 来源条目：调度控制台使用说明
- 控制台入口：路网大屏 `/`
- 当前版本边界：调度控制台已经具备演示级状态监控、AI 建议、手动相位切换和应急绿波展示能力。真实调度闭环仍需审批、权限、审计和后端任务持久化接入。
- 相关组件：
  - `TrafficStats`
  - `AlertPanel`
  - `MapRoadNetwork`
  - `SignalControlPanel`
  - `EmergencyPanel`
  - `CompareCharts`
  - `AiAssistant`

## 控制台定位

调度控制台用于让调度人员在同一个页面中完成“看态势、看告警、看路口、看建议、发起演示控制、查看效果”的闭环。

当前控制台以路网大屏为主体，适合演示以下工作：

1. 监控城市路网运行状态。
2. 识别拥堵路段和异常路口。
3. 查看 AI 对当前路口的配时建议。
4. 暂停或启动 AI 自适应控制演示状态。
5. 手动切换当前路口相位。
6. 触发应急车辆和绿波通道演示。
7. 处理实时告警。
8. 调用智能体生成解释和建议。

## 控制台关键指标

“交通统计”面板提供 10 项全局指标：

| 指标 | 含义 |
|---|---|
| 当前总车流量 | 当前路网总流量，单位为辆/h |
| 平均通行速度 | 当前路网平均速度，单位为 km/h |
| 平均等待时间 | 当前路网平均等待时间，单位为秒 |
| 当前拥堵指数 | 0 到 100 的综合拥堵指标，数值越高越拥堵 |
| 拥堵路段数量 | 当前拥堵指数较高的路段数量 |
| 已优化路口 | 当前可参与优化或在线运行的路口数量 |
| 应急车辆 | 当前路网上的应急车辆数量 |
| 设备在线率 | 信号机或路口设备在线比例 |
| 今日告警数量 | 当日累计告警数量 |
| 绿波执行次数 | 当前或当日绿波执行展示次数 |

## 调度工作流程

### 1. 确认系统状态

进入路网大屏后，先查看右上角数据来源。

- 如果显示“数据库数据”，说明当前数据来自后端。
- 如果显示“本地演示”，说明后端数据不可用，当前只能进行演示和知识库问答。
- 如果显示“连接中”，等待系统自动重试。

### 2. 判断路网态势

查看“交通统计”和中央路网。

重点关注：

- 拥堵指数是否升高。
- 平均等待时间是否异常。
- 拥堵路段数量是否增加。
- 设备在线率是否下降。
- 是否存在应急车辆或绿波任务。

### 3. 定位问题路口

可通过以下方式定位问题：

- 在实时告警面板中点击告警。
- 在路网中选择拥堵或异常路口。
- 在智能体中询问“哪个路口最拥堵？”。

选中路口后，右侧“AI 信号控制”面板会显示当前路口详情。

### 4. 查看 AI 调度建议

在“AI 信号控制”面板中阅读“AI 决策建议”。

系统会根据当前路口状态生成不同建议：

| 状态 | 建议特征 |
|---|---|
| 设备故障 | 提示人工介入、派单巡检、降级控制 |
| 设备离线 | 提示 AI 决策暂停、等待设备恢复 |
| 严重拥堵 | 建议延长当前绿灯、提前下游窗口、上游限流 |
| 中度拥堵 | 建议维持配时并持续监控 |
| 轻度拥堵或畅通 | 建议标准配时运行 |

### 5. 执行演示控制

面板提供三个按钮：

| 按钮 | 当前作用 | 注意事项 |
|---|---|---|
| 启动 AI 自适应控制 | 设置前端 `aiEnabled=true`，生成操作提示 | 当前不是生产控制指令 |
| 暂停 AI 控制 | 设置前端 `aiEnabled=false`，保留当前配时演示状态 | 当前不是生产停控指令 |
| 手动切换相位 | 对当前路口切换到下一相位 | 当前只作用于前端演示数据 |

所有真实控制动作都应在接入后端任务、审批、权限和审计后执行。

### 6. 处理告警

在“实时告警”面板处理告警。

1. 查看紧急、严重、预警和提示级别。
2. 点击告警查看并确认。
3. 如果告警关联路口，系统会同步选中路口。
4. 需要批量处理时点击“全部确认”。

确认告警只表示调度人员已看到并处理面板状态，不代表故障已经恢复。

### 7. 查看效果对比

底部“控制效果对比”用于展示传统控制与 AI 控制的指标差异，例如平均等待时间、平均速度、排队长度、应急通行时间等。

调度人员可以根据该区域判断当前演示策略是否改善通行效率。

## 推荐调度口径

当系统处于演示态时，调度建议应使用以下表达：

- “系统建议延长当前相位，状态为待人工确认。”
- “当前为演示数据，需以后端仿真或真实路口数据复核。”
- “该操作仅改变前端演示状态，不直接下发真实信号机。”

不应使用以下表达：

- “系统已经真实控制路口。”
- “AI 已自动抢占全部信号。”
- “告警确认表示故障已修复。”

## 适合回答的问题

- 调度控制台在哪里？
- 调度员如何判断拥堵？
- 调度建议在哪里确认？
- 手动切换相位会做什么？
- 当前调度控制是否是真实控制？
- 如何从告警定位到路口？
- 控制效果对比看哪些指标？
