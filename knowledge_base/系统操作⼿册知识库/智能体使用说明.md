# 智能体使用说明

## 入口与用途

在路网大屏右下角点击“AI 决策”打开窗口。可询问仿真状态、拥堵原因、路口或道路详情、控制决策、日志、知识库和应急事件。点击“新对话”会清空前端保存的 `sessionId`、`conversationId` 和当前消息。

## 当前调用链

前端只调用 `POST /api/v1/agent/chat`。请求包含 `message`，并可携带仿真会话 `sid`、Agent 会话 `conversationId`、兼容会话 `sessionId` 和页面 `context`。

Spring Boot 负责创建或读取会话、保存消息、调用模型规划白名单工具、执行工具并记录审计。实时状态工具读取内存最近仿真帧；历史和日志工具读取数据库。

## 前端状态

| 状态 | 含义 |
|---|---|
| 后端 Agent 待连接 | 尚未完成本轮调用 |
| 后端 Agent 在线 | 后端返回非兜底响应 |
| Agent 兜底响应 | 后端返回标记为 fallback 的响应 |
| 后端 Agent 异常 | 请求失败或响应无有效内容 |

前端当前没有本地规则问答兜底。失败时会显示后端错误，并提示检查后端及模型配置。

## 控制边界

Agent 工具白名单不执行信号控制、应急调度或权限修改。涉及控制的问题只能给出有证据的分析与建议。工具失败或没有真实数据时必须明确说明，不能编造状态或声称动作已执行。
