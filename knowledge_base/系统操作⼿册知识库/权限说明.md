# 权限说明

## 当前代码事实

- 前端提供 `/login` 和 `/register` 页面。
- 后端提供密码登录、验证码登录、注册和发送验证码接口。
- 登录成功返回临时 UUID token，但当前没有全局鉴权拦截器。
- 业务接口和前端按钮没有按角色执行授权控制。
- 项目没有实现观察员、调度员、应急管理员、系统管理员等角色权限矩阵。

因此，当前登录功能不能证明调用者有权执行仿真、应急调度、状态修改或 Agent 工具调用。页面可见、按钮可点击也不代表具备生产权限。

## 当前操作边界

- AI 启停和手动相位切换只修改前端状态。
- 应急调度优先调用仿真接口；没有仿真会话或接口失败时回落到本地模拟。
- Agent 白名单工具用于查询和诊断，不直接下发信号控制。
- Agent 会话、消息和工具调用可以写入审计数据，但系统尚无完整的用户授权和控制审批链路。

在鉴权、授权和审批实现前，不应把本系统用于未经人工监管的真实信号控制，也不应虚构角色、审批人或权限范围。
