# 绿波任务恢复流程

## 操作

1. 在“应急绿波控制”面板确认当前为“绿波执行中”。
2. 查看路线、车辆和已放行路口进度。
3. 点击“恢复普通控制”。
4. 确认面板转为已完成，并查看恢复提示告警。

按钮仅在 `systemMode === 'emergency'` 时可用。

## 当前代码行为

恢复操作调用前端 store，并执行：

- `systemMode` 改为 `normal`。
- `greenWaveActive` 改为 `false`。
- ETA 重置为 8。
- `activeGreenWaveIndex` 改为 -1。
- 若能按应急车辆业务 ID 找到前端车辆，则把其类型改为普通车辆并重设演示速度。
- 生成“已恢复普通控制模式，应急绿波通道关闭”前端告警。

当前恢复操作没有调用后端停止应急任务或恢复信号策略的接口。因此它只表示前端演示状态恢复，不能声称真实信号优先已经关闭，也不能作为任务完成或审计记录。
