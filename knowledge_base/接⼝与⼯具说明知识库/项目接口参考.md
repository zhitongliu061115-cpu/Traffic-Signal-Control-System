# 项目接口参考

本文只记录当前项目接口。字段、DTO、请求示例、错误码和完整索引以 `backend/docs/API_GUIDELINES.md` 为准。

## 调用边界

- 前端只调用 Spring Boot 主后端。
- Spring Boot 内部调用 Python CityFlow 和 Traffic-R；前端不得直连内部服务。
- Spring Boot 普通 REST 响应使用 `success`、`message`、`data`。
- 静态路网使用 REST；实时仿真帧使用 WebSocket。
- 当前没有全局鉴权拦截器。登录返回的临时 token 尚未用于保护业务接口。

## 主要 REST 接口

| 方法 | 路径 | 用途 |
|---|---|---|
| `POST` | `/api/auth/send-captcha` | 发送邮箱验证码 |
| `POST` | `/api/auth/login` | 用户名密码登录 |
| `POST` | `/api/auth/captcha-login` | 邮箱验证码登录 |
| `POST` | `/api/auth/register` | 注册账号 |
| `GET` | `/api/v1/dashboard/bootstrap` | 路网大屏展示数据 |
| `GET` | `/api/v1/data-analysis/bootstrap` | 数据分析页展示数据 |
| `GET` | `/api/v1/database/status` | 数据库及核心表状态 |
| `GET` | `/api/v1/intersections` | 查询旧展示路口 |
| `GET` | `/api/v1/intersections/{code}` | 按 code 查询旧展示路口 |
| `PATCH` | `/api/v1/intersections/{code}/status` | 更新旧展示路口状态 |
| `GET` | `/api/v1/scenes/{sceneId}/roadnet` | 获取静态路网 |
| `POST` | `/api/v1/simulations` | 创建仿真会话 |
| `POST` | `/api/v1/simulations/{sid}/start` | 启动仿真 |
| `POST` | `/api/v1/simulations/{sid}/pause` | 暂停仿真 |
| `POST` | `/api/v1/simulations/{sid}/stop` | 停止仿真 |
| `POST` | `/api/v1/simulations/{sid}/dispatch` | 调度应急车辆 |
| `POST` | `/api/v1/agent/chat` | Agent 统一聊天入口 |

`/api/v1/runtime/**` 用于数据库历史与复盘查询。当前仿真状态应使用 `/api/v1/agent/tools/**` 中读取内存实时帧的工具接口。Agent 会话、消息、审计接口的完整列表见权威接口文档。

## WebSocket

连接路径：`/ws/v1/simulations/{sid}`。

前端处理 `sim.frame`、`sim.status`、`sim.error` 和 `control.decision`。其中 `control.decision` 只表示后端已生成并提交动作，控制效果以后续 `sim.frame.data.signals` 为准。

## 数据与安全

- JSON 字段使用 `camelCase`；数据库表和字段使用 `snake_case`。
- 场景、道路和路口 ID 保留 CityFlow 原始命名。
- 知识库不得保存 API Key、Token、Cookie、认证头、个人信息或未脱敏轨迹。
- 服务地址从部署配置读取，不在知识库固化公网或内网地址。
- 接口、DTO、WebSocket、认证头或服务地址变化时，必须更新 `backend/docs/API_GUIDELINES.md`。
